## Featured Projects

### 🛡️ SEA-GUARD
A multi-agent reinforcement learning (MARL) project focused on autonomous surface vessel capture-the-flag simulation. 
* **Core Features:** Implemented curriculum training, league-style opponent progression, team-consistent reward shaping, and cooperative coordination behaviors.
* **Technologies:** Python, PyTorch, PPO, Unity (for visualization).

### ⚡ Quantum Strike
A mixed-reality action game project designed for spatial computing.
* **Core Features:** Built interactive gameplay loops, weapon system enhancements, HUD layouts, and integrated **K2B2**—an interactive AI companion that responds in natural language with synthesized voice output.
* **Technologies:** Unity, C#, Mixed Reality Toolkit, OpenAI API, ElevenLabs.

### 🔍 Inquiry Engine
An AI-driven research pipeline for automated document analysis and fact-checking.
* **Core Features:** Ingests transcripts, extracts core claims, preserves source-grounded citations, and constructs a structured evidence layer for transparent reasoning.
* **Technologies:** Python, LLMs, NLP, Vector DBs, MLOps.

### 🤖 Acera AI
A desktop agentic assistant designed to showcase local multimodal control.
* **Core Features:** Integrates natural voice interaction, structured command parsing, and experimental gesture-control features.
* **Technologies:** Python, Speech Recognition, OpenCV, Local LLMs.
